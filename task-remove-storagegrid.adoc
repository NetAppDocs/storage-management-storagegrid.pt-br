= Remover um sistema StorageGRID
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Remova um sistema StorageGRID se não quiser mais gerenciá-lo no NetApp Console.

.Função necessária do NetApp Console :
Administrador de armazenamento. link:https://docs.netapp.com/us-en/console-setup-admin/reference-iam-predefined-roles.html["Saiba mais sobre as funções de acesso do NetApp Console"^].

Remover o sistema do Console não afeta o sistema StorageGRID .  Você pode adicioná-lo novamente ao Console a qualquer momento.

.Passos
. No menu de navegação, selecione *Armazenamento > Gerenciamento*.
. Na página *Sistemas*, abra o sistema StorageGRID .
. Na visualização padrão, selecione o ícone de menu e selecione *Remover do espaço de trabalho*.
. Selecione *Remover* para confirmar.
